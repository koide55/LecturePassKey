# passkey

Passkey learning repository for lectures and hands-on exercises.

## Repository layout

- `slides/`
  - Markdown source for lecture materials
- `app/`
  - Hands-on exercise software and notes
- `docs/`
  - Supplemental documents

## First learning materials

- Slide index: [slides/README.md](slides/README.md)
- Introductory notes: [slides/lecture01_passkeys_foundation.md](slides/lecture01_passkeys_foundation.md)

## Goal

This repository is designed to help learners understand:

- why passwords are hard to defend
- how passkeys improve phishing resistance
- how WebAuthn and FIDO2 fit together
- what needs to be implemented on the server and client
- how to design a safe hands-on lab
